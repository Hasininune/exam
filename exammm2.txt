minikube commands:
✅ SECTION 1 — PRE-REQUISITE CHECK COMMANDS

Run these first to confirm everything is installed.

systeminfo | findstr /i "Virtualization"
docker --version
docker ps
minikube version
kubectl version --client

✅ SECTION 2 — MINIKUBE STARTUP COMMANDS

Start Minikube using Docker driver:

minikube start --driver=docker
minikube status
kubectl get nodes


Open Kubernetes Dashboard:

minikube dashboard

✅ SECTION 3 — DEPLOY NGINX ON MINIKUBE
Create a deployment
kubectl create deployment mynginx --image=nginx
kubectl get deployments
kubectl get pods

✅ SECTION 4 — EXPOSE DEPLOYMENT AS A SERVICE
kubectl expose deployment mynginx --type=NodePort --port=80
kubectl get svc


Get the URL and open NGINX in browser:

minikube service mynginx


If above doesn't open browser:

minikube service mynginx --url


Alternate (port-forward):

kubectl port-forward svc/mynginx 8081:80


Then visit: http://localhost:8081

✅ SECTION 5 — SCALE THE DEPLOYMENT
kubectl scale deployment mynginx --replicas=4
kubectl get pods

✅ SECTION 6 — DEBUG / LOGS (If asked)
kubectl describe pod <pod-name>
kubectl logs <pod-name>
kubectl get events --sort-by=.metadata.creationTimestamp

✅ SECTION 7 — CLEANUP COMMANDS
kubectl delete service mynginx
kubectl delete deployment mynginx
minikube stop
minikube delete

✅ SECTION 8 — NAGIOS DOCKER COMMANDS
Pull Nagios Image
docker pull jasonrivers/nagios:latest

Run Nagios Container
docker run --name nagiosdemo -d -p 8888:80 jasonrivers/nagios:latest

Check Running Container
docker ps
docker logs nagiosdemo

Access Nagios Dashboard

Open in browser:

http://localhost:8888


Login:

Username: nagiosadmin

Password: nagios

✅ SECTION 9 — STOP/REMOVE NAGIOS
docker stop nagiosdemo
docker rm nagiosdemo


Optional (remove image):

docker rmi jasonrivers/nagios:latest

        stage('Deploy to Tomcat') {
            steps {
                echo "Deploying WAR to Tomcat..."
                deploy adapters: [tomcat9(
                    credentialsId: 'tomcat-creds',
                    path: '',
                    url: 'http://localhost:8081'
                )],
                contextPath: '/mavenweb',
                war: 'target/*.war'

--------------pipeline--------------
node {
  stage('Checkout') {
    git branch: 'master', url: 'https://github.com/Hasininune/mavenwebproject.git'
  }

  stage('Build') {
    bat 'mvn clean package'
  }

  stage('Test') {
    bat 'mvn test'
  }

  stage('Deploy') {
    // get username/password from Jenkins credentials
    withCredentials([usernamePassword(credentialsId: 'tomcat-creds',
                                      usernameVariable: 'TOMCAT_USER',
                                      passwordVariable: 'TOMCAT_PASS')]) {
      echo "Deploying WAR via Tomcat Manager (curl)..."
      // ensure the WAR path matches what your build produces:
      def warPath = 'target\\maven-webapp.war'

      // curl -T to upload and deploy, update=true to replace existing app
      bat """
      curl -u %TOMCAT_USER%:%TOMCAT_PASS% -T ${warPath} "http://localhost:8081/manager/text/deploy?path=/mavenweb&update=true"
      """
    }
  }
}
